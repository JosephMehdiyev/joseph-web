Homepage of Joseph

This is the homepage of Joseph's (Yusif Mehdiyev) website, a site fully written and designed by Joseph for educational and professional purposes.

Using the navigation bars of the page, the reader may visit various pages of the website. Information about my CV, Projects and Publications can be found in the navigation pages. I recommend reading the website with a laptop/desktop since telephone version of the website is a bit clunky. Be warned!

If you want to know more about how this site is written, you may visit my github page or read the site's This website page. To be precise and simple, The site is written with pure javascript, html and css. I use jekyll, a Static Site generator written in Ruby, to generate my html pages. Note that the website is still in development so there may be aesthetical or functional issues when you read the pages. As written above, I recommend using the desktop/laptop version of the site for the best experience. I am not that interested in web development and I have other projects in my mind, so this site probably stay the same for centuries.
